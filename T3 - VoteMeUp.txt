1. Цель проекта


Создать веб-сайт, на котором каждый зарегистрировавшийся пользователь сможет предложить решение любой проблемы в масштабе города, района, страны. Проголосовать за или против уже предложенных решений. К каждому предложению зарегистрированные пользователи смогут оставлять комментарии со своими мыслями о предложенном решении. 
В зависимости от выбранного масштаба, предлагаемое решение должно набрать минимально заданное число голосов “За”, до даты окончания голосования для передачи этого предложения в органы законодательной власти.

2.5. Пользователи (User). Для того чтобы создавать новые предложения на сайте и иметь возможность голосовать и оставлять комментарии, пользователь должен зарегистрироваться. При этом зарегистрированный пользователь может получить одну из заданных ролей на сайте. Каждая роль отличается набором привилегий (допусков), которые определяют возможности пользователей.
Основные роли и привилегии предусмотренные на сайте:
1. Администратор
        -  активация нового зарегистрированного пользователя
        - запрет на создание пользователем новых предложений
2. Обычный пользователь
        - создание новых предложений
        - создание новых комментариев к предложению
        - голосовать “за” предложения
        - голосовать “против” предложения
3. Контент менеджер
        - запрет на создание новых комментариев пользователю к предложениям
        - “удаление” некорректных комментариев к предложению

3. Основные бизнес-правила для Пользователей
3.1. Пользователь сайта, который не залогинился в системе, может только просматривать предложения и  комментарии. 
3.2. При регистрации на сайте, пользователь должен заполнить форму с указанными обязательными полями (предусмотреть CAPCHA на форме регистрации). После нажатия кнопки регистрации, новому пользователю должно прийти письмо на указанный им электронный адрес с кодом подтверждения. который должен активировать его учетную запись.
3.3. Для зарегистрированных пользователей доступен персональный раздел с дополнительными настройками (подписки на обновления, быстрыми отборами по его предложениям, комментариям и т.д.)
3.4. Зарегистрированный пользователь может вносить предложения на сайте (должна быть кнопка быстрого доступа)
3.5. Зарегистрированный пользователь может голосовать “За” и “Против” Предложений на сайте. За Предложение он может отдать только один голос.
3.6. Зарегистрированный пользователь может оставлять комментарии к Предложениям. Может комментировать другие комментарии. Может отметить понравившийся Комментарий “Likом”, но только один раз.
